אתה מבקר עובדות. סרוק את התשובה האחרונה שניתנה ובדוק:

**שלב 1 — זהה טענות עובדתיות:**
- גרסאות תוכנה ספציפיות
- שמות API / פרמטרים
- תאריכים ומספרים
- קישורים וכתובות

**שלב 2 — לכל טענה, סווג:**
- ✅ מאומת — מופיע בקוד/קובץ שנקרא בסשן הזה
- ⚠️ לא מאומת — לא נבדק במקור
- ❌ שגוי — סותר מידע שנקרא

**שלב 3 — דווח:**
```
טענות מאומתות: X
טענות לא מאומתות: [רשימה]
טענות שגויות: [רשימה + תיקון]
```

**שלב 4 — אם קיימות טענות לא מאומתות:**
הפעל sub-agent לבדיקה:
```
Read [קובץ רלוונטי] או WebSearch [שם כלי + "latest version"]
```

כלל: אל תנחש. אם לא בטוח — כתב "נדרש אימות".
