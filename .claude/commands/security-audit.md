בצע סריקת אבטחה מלאה. הפעל לפי הסדר:

**קלט:** $ARGUMENTS (נתיב קובץ/תיקייה — ברירת מחדל: כל הפרויקט)

**שלב 1 — מפה את קבצי הקלט:**
Glob: `**/*.py`, `**/*.js`, `**/*.ts`, `**/*.sh`, `**/*.env*`, `**/*.json`

**שלב 2 — לכל קובץ, בדוק:**

| בעיה | מה לחפש |
|------|----------|
| Command injection | `subprocess`, `exec`, `eval`, `os.system` עם משתני קלט |
| SQL injection | f-strings/concatenation בתוך queries |
| XSS | הדפסת קלט משתמש ישירות ל-HTML |
| Credentials חשופים | hardcoded passwords, API keys, tokens |
| קלט לא מסונן | קלט משתמש שעובר ישירות לפונקציות מערכת |
| הרשאות קבצים | `chmod 777`, קבצי config עם הרשאות רחבות |
| Secrets ב-git | `.env` files, private keys ב-repo |

**שלב 3 — דווח בפורמט:**
```
קריטי: [רשימה + שורה + הסבר]
גבוה: [רשימה]
בינוני: [רשימה]
נקי: [קבצים שנסרקו ועברו]
```

**שלב 4 — תקן אוטומטית** רק ממצאים קריטיים. השאר — דווח בלבד.
