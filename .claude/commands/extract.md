אתה מומחה כריית ידע. קבל טקסט גדול ותפיק ממנו ידע מובנה.

**קלט:** $ARGUMENTS (נתיב קובץ או טקסט ישיר)

**פרוטוקול:**

שלב 1 — אם זה נתיב קובץ, קרא אותו. אם זה URL של יוטיוב, הרץ:
```
python youtube_transcribe.py <url> --model base --output /tmp/transcript.txt
```
ואז קרא את הפלט.

שלב 2 — סרוק את הטקסט וחלץ:

```
## עובדות מרכזיות
- [עובדה מדויקת + מקור בטקסט]

## החלטות והמלצות
- [החלטה/המלצה + הסבר קצר]

## מושגים מפתח
- [מושג]: [הגדרה מדויקת מהטקסט]

## אזהרות וסיכונים
- [אם קיימים]

## שאלות פתוחות
- [מה הטקסט לא מסביר]
```

שלב 3 — שמור לקובץ: `memory/knowledge-[שם-נושא]-[תאריך].md`

שלב 4 — עדכן `memory/active-context.md` עם שם הקובץ שנוצר.

**כלל ברזל:** אל תוסיף מידע שלא קיים בטקסט. אם משהו לא ברור — ציין "לא מפורש בטקסט".
