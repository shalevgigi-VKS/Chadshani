# Project Index — קסם הידיעה

## Goal
Produce short Hebrew educational videos (40–50 sec) for children aged 4–10.
Each episode explains one life-habit topic using colorful characters, slow pacing, and warm tone.

## Core Identity
- Name: קסם הידיעה
- Language: Hebrew only
- Duration: 40–50 seconds per episode
- Target audience: Ages 4–10
- Characters: One child character + one animal companion (fixed duo per series, or varied)
- Tone: Flexible per episode — calm/storytelling / energetic / playful-smart
- Format: Flexible per episode — animation / static+voiceover / animation+real voiceover

## Modules

| # | Module | Purpose |
|---|---|---|
| 01 | Character System | Define the child + animal duo, visual identity |
| 02 | Episode Template | The reusable script + scene structure per episode |
| 03 | Production Pipeline | Tools, workflow, and output format |
| 04 | Topic Bank | List of episode topics with priority order |
| 05 | Voiceover System | Hebrew narration rules, pacing, tone per episode |
| 06 | Visual Style Guide | Color, aesthetic, animation/static rules |
| 07 | Episode 01 — Teeth Brushing | First full episode: script + scenes + production |
| 08 | Web Interface | Simple topic-in → script-out tool |

## Dependencies
- Module 01 must be complete before 02, 06, 07
- Module 02 must be complete before 05, 07
- Module 03 is independent — can run in parallel
- Module 04 is independent — can run in parallel
- Module 07 requires 01 + 02 + 05 + 06
- Module 08 is independent — requires only Anthropic API key
