# Module 01 — Character System

## Objective
Define the permanent visual and personality identity of the two main characters.

## Scope
Child character + animal companion. Names, personalities, visual description, relationship dynamic.

## Requirements
- Child character: relatable, curious, ages 5–7 appearance
- Animal companion: warm, knowledgeable, enthusiastic (not a teacher — a friend)
- Both characters present in every episode
- Visual style: colorful, rounded shapes, large expressive eyes
- Gender: default gender-neutral or alternating

## Character Profiles to Define

### Child Character
- Name: [to be chosen]
- Age appearance: 5–6 years
- Personality: curious, asks questions, tries the habit at end of each episode
- Visual: rounded face, big eyes, bright clothing, warm skin tone
- Role in episode: asker, learner, demonstrator

### Animal Companion
- Species: [to be chosen — suggestions: owl, rabbit, small dragon, friendly bear]
- Name: [to be chosen]
- Personality: enthusiastic, warm, never lectures, uses simple words
- Visual: same rounded style, matches child character's color palette
- Role in episode: explainer, encourager

## Relationship Dynamic
- Equal friends — not teacher/student
- Animal is not smarter, just happens to know this one thing
- Child teaches back at the end (demonstrates the habit)

## Visual Consistency Rules
- Same character design used across ALL episodes
- Color palette locked after Module 06
- No design changes mid-series

## Output of This Module
A written character sheet per character:
- Name
- Species (for animal)
- 5 personality traits
- Visual description (colors, features, clothing/markings)
- 3 sample lines of dialogue that capture their voice
