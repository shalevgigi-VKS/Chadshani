# Module 06 — Visual Style Guide

## Objective
Define the color palette, aesthetic, and visual rules for all episodes.

## Core Aesthetic
- Warm, bright, rounded
- Reference: Israeli children's book illustration style
- No sharp edges, no dark tones
- Everything feels soft and safe

## Color Palette
- Backgrounds: warm cream, soft sky blue, light green, warm yellow
- Characters: saturated but not neon — think Bluey palette
- Avoid: black backgrounds, dark grey, heavy shadows

## Character Visual Rules
- Large eyes (40% of face)
- Round head, soft body proportions
- Bright, single-color clothing
- Animal: same rounded style, not realistic

## Scene Rules
- 2–3 background elements maximum per scene (not cluttered)
- Habit object always visible and large in Act 2 scenes
- Child character always smiling in Act 3

## Text on Screen
- Hebrew only
- Large, bold, rounded font (e.g. Varela Round or similar)
- High contrast: dark text on light background or light text on colored shape
- Key message appears on screen during Act 3

## Animation Style (if animated)
- Simple movements: bounce, slide, pop-in
- No complex camera movements
- Character reactions: nod, smile, small jump — not full body animation
- 2D flat style preferred over 3D

## Consistency Rule
- Same character colors every episode — locked after first episode
- Background style consistent — warm and bright always
- Font never changes
