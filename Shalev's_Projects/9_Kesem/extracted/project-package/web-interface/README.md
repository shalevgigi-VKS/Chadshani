# ממשק האינטרנט — קסם הידיעה

## שימוש מיידי
1. פתח את `index.html` בדפדפן (Chrome / Safari / Firefox)
2. הזן מפתח API של Anthropic (קבל בחינם: https://console.anthropic.com)
3. כתוב נושא בעברית
4. לחץ "צור תסריט"
5. קבל תסריט מלא — העתק או הורד

## מה מקבלים
- כותרת הפרק
- המסר המרכזי
- תסריט 3 מערכות מלא
- פירוט סצינות עם זמנים
- הוראות קריינות
- רשימת ייצור עם checkboxes

## דרישות
- דפדפן מודרני
- מפתח API של Anthropic (https://console.anthropic.com)
- חיבור אינטרנט

## פרטיות
המפתח אינו נשמר בשום מקום — נמחק בסגירת הלשונית.

## תמיכה
אם הממשק לא עובד — בדוק שהמפתח תקין ושיש חיבור לאינטרנט.
