# Handoff Notes — קסם הידיעה

## Ambiguities

### Character Names
Not yet defined. Module 01 must decide:
- Child character name (Hebrew, gender-neutral preferred)
- Animal species and name
- Are these fixed across all episodes or can vary?

### Production Budget
Tools chosen in Module 03 depend on budget level:
- Zero budget: Canva + ElevenLabs + CapCut
- Low budget: Runway / Kling AI + professional voiceover recording
- Medium budget: custom animation studio or freelance animator
This decision must be made before Module 03 is built.

### Distribution Channel
Where will episodes be published? (YouTube / Instagram Reels / WhatsApp / private)
Affects aspect ratio (16:9 vs 9:16) and output resolution.
Must be decided before Module 03.

### Voiceover — Real vs AI
Will narration be recorded by a real person (parent, actor) or AI-generated Hebrew voice?
Affects Module 05 significantly.

## Missing Information
- Final character designs (needed for Module 06 and all production)
- Distribution platform decision (needed for Module 03)
- Voiceover approach decision (needed for Module 05)

## Risks
- Hebrew AI voiceover quality is inconsistent across tools — test before committing
- AI animation tools (Runway, Kling) may struggle with consistent character appearance across frames
- 40–50 sec is very short — overwriting the script is the most common failure

## No-Assumption Zones
- Do not assume animation tool can maintain character consistency without testing
- Do not assume AI Hebrew voice sounds natural without listening test
- Do not assume 4-year-old comprehension without reading script aloud to a real child
