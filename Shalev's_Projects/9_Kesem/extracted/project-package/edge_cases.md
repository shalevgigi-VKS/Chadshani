# Edge Cases — קסם הידיעה

## Duration Overrun
- Script exceeds 50 seconds when narrated slowly
- Resolution: cut Act 2 to 2 explanations maximum. Never cut Act 3.

## Topic Too Complex for Age
- Explanation requires more than 4 sentences
- Resolution: simplify to ONE reason only. Children do not need full science.

## Character Conflict
- Animal character's personality contradicts episode tone
- Resolution: tone of episode overrides character default personality

## Voiceover Mismatch
- Hebrew text sounds unnatural when read aloud
- Resolution: rewrite for spoken Hebrew, not written Hebrew. Read aloud before finalizing.

## Visual Tool Limitation
- Chosen production tool cannot render the intended scene
- Resolution: simplify scene, not concept. Concept integrity is mandatory.

## Age Range Gap
- Content works for 8–10 but is too complex for 4–5
- Resolution: always default to 4–5 comprehension level. Older kids self-adjust.

## Topic Repeat
- Two episodes cover same habit from different angles
- Resolution: merge into one episode or delay second until 10+ episodes exist

## TTS Voice Quality
- Hebrew TTS sounds robotic or unclear on a specific browser
- Resolution: interface must test TTS before starting animation; if quality is below threshold, display warning and offer to continue or cancel

## TTS Timing Mismatch
- Synthesized speech runs longer or shorter than the scene's allotted duration
- Resolution: scene duration adapts to actual TTS length — script duration estimates are guidelines, not hard locks; total episode may stretch to 55 seconds maximum

## Background Music Unavailable
- Music file fails to load (network error, CDN unavailable)
- Resolution: continue without music; voiceover and animation play normally; do not block video production

## Browser TTS Language Support
- Browser does not support Hebrew TTS voice
- Resolution: display a clear message listing browsers that support Hebrew TTS (Chrome on most platforms); do not attempt playback with wrong language voice

## Child Character Gender
- Character gender creates exclusion for part of audience
- Resolution: default to gender-neutral name and visual. Or alternate per episode.
